#Instalação

###Instalar Node.Js e NPM
`$ sudo apt-get install nodejs`

###Verificar Instalação
`$ node -v `
**v13.3.0**

`$ npm -v `
**6.13.1**
###Instalar Pacotes
`$ npm install `

#Executar aplicação
`npm start`

**Acessar no navegador link indicado no console**
